### 2.4 Repair any remaining canonical URL defects

For every page where `<link rel="canonical">` does not match the page’s own URL, correct it to the exact URL the page lives at. Pattern: `https://glee-fully.tools/{path-from-root}/` (with trailing slash).

```bash
# Automated canonical check and fix
python3 - <<'EOF'
import os, re
from pathlib import Path

base = "https://glee-fully.tools"
for path in Path('.').rglob('index.html'):
    # Skip root
    if str(path) == 'index.html':
        continue
    url_path = '/' + str(path.parent).lstrip('./') + '/'
    expected = f"{base}{url_path}"
    content = path.read_text(encoding='utf-8')
    current = re.search(r'<link rel="canonical" href="([^"]+)"', content)
    if current and current.group(1) != expected:
        print(f"FIX: {path} | was: {current.group(1)} | should be: {expected}")
        new_content = content.replace(current.group(0), f'<link rel="canonical" href="{expected}"')
        path.write_text(new_content, encoding='utf-8')
EOF
```
